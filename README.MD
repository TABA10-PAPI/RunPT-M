# 🏃 RunPT Frontend

AI 기반 러닝 코칭 및 신체 배터리 제공 서비스를 위한 프론트엔드 프로젝트.

사용자의 러닝 데이터를 시각화하고, AI 코칭 기능과 운동 기록 관리 기능을 제공하는 러닝 플랫폼.

---

## 📌 프로젝트 소개

RunPT는 사용자의 운동 데이터를 기반으로
러닝 기록 관리, AI 기반 신체 상태 분석, AI 코칭 기능 등을 제공하는 서비스.

프론트엔드는 React Native 기반으로 개발했으며
직관적인 UI와 모바일 친화적인 UX를 중심으로 구현.
